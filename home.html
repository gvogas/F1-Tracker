About

  • Powered by the free, public OpenF1 API — no API key needed
  • Live timing tower with differential DOM updates (no full re-renders)
  • Canvas-based track position map drawn from GPS telemetry data
  • Race replay at selectable speeds (1×–10×) with seek scrubber
  • Rate-limit aware — exponential backoff on 429 responses
  • User favorites stored in cookies — no server required
  • Fully responsive — works on mobile, tablet, and desktop
